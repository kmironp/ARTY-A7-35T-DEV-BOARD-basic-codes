Here are simple software compilation setup for
- Arm Compiler 5
- Arm Compiler 6
- gcc
The setups in this directory is only used for illustration 
of minimum compilation setup and are not used for simulation.
Instead, the compilation setup in ../../simenv/testcode is used.
That contains the setup which supports multiple toolchains.

The cmsis_include directory contains CMSIS-CORE header files 
for Cortex-M processors. For convenience these files are 
included here, but you should download the latest CMSIS-CORE
header files from
  https://github.com/ARM-software/CMSIS_5/releases
